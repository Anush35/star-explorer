# star-explorer
# ✦ Star Constellation Explorer

An interactive night sky website where you can explore 6 constellations, their Greek mythology, real star names, and seasonal visibility — all built with pure HTML, CSS, and JavaScript. No frameworks, no libraries.

🔭 **Live site → [anush35.github.io/star-explorer/star-explorer.html](https://anush35.github.io/star-explorer/star-explorer.html)**

---

## Preview

> A dark animated night sky with twinkling stars. Click any constellation to reveal its mythology, brightest stars, and the months it's visible from Earth.

---

## Features

- 🌌 6 real constellations — Orion, Ursa Major, Cassiopeia, Leo, Scorpius, Gemini
- 📖 Greek mythology story for each constellation
- ⭐ Real star names with magnitude (brightness) data
- 📅 Filter by month to see which constellations are in season
- ✨ Twinkling star animations with glow effects
- 🖱️ Hover to highlight, click to explore
- 📱 Works on mobile (touch support)
- 🚀 No frameworks — pure HTML, CSS, JavaScript

---

## How to run locally

1. Download or clone this repo
2. Open `star-explorer.html` in any browser
3. That's it — no server or install needed

```bash
git clone https://github.com/Anush35/star-explorer.git
cd star-explorer
# open star-explorer.html in your browser
```
